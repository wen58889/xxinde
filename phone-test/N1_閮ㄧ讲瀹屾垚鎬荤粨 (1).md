## 三、配置文件

### 3.1 Klipper配置

**路径**：`~/printer_data/config/printer.cfg`

**机械参数**：
- X轴行程：0~150mm
- Y轴行程：0~150mm
- Z轴行程：0~50mm
- Z安全高度：30mm
- XY最大速度：F9000 (150mm/s)
- Z最大速度：F4800 (80mm/s)

**⚠️ 注意**：MCU串口路径需根据实际硬件确认：
```
serial: /dev/serial/by-id/usb-Klipper_rp2040_504450593027961C-if00
```

### 3.2 Moonraker配置

**路径**：`~/printer_data/config/moonraker.conf`

```ini
[server]
host: 0.0.0.0
port: 7125
klippy_uds_address: /tmp/klippy_uds

[authorization]
trusted_clients:
    0.0.0.0/0
    ::1/128
cors_domains:
    *
```

### 3.3 go2rtc配置（按需取帧）

**路径**：`/etc/go2rtc/go2rtc.yaml`

```yaml
api:
  listen: ":1984"

streams:
  camera0:
    - exec:ffmpeg -f v4l2 -input_format mjpeg -video_size 1280x720 -i /dev/video1 -frames:v 1 -f image2pipe -
```

**截图接口**：`GET http://192.168.5.101:1984/api/frame.jpeg?src=camera0`

## 七、快速命令参考

```bash

# 重启服务
systemctl restart klipper
systemctl restart moonraker
systemctl restart go2rtc

# 查看日志
journalctl -u moonraker -f
journalctl -u klipper -f
journalctl -u go2rtc -f

# 发送G-code

  -H "Content-Type: application/json" \
  -d '{"script":"G28"}'
```
